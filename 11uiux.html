PharmaNeeds

E-Pharmacy App Onboarding

ABOUT THE PROJECT

This project was done as an individual assignment. The design brief was to design screens for an e-pharmacy app that has been developed by an eCommerce company for customers to buy medication online. The app eases the life of customers in many ways including home delivery of medication, online payment, privacy in buying medications, etc.

SCOPE OF WORK

The screens to be designed were as follows:
• Onboarding / introduction screens. The intro screens should be able to convince the customer about the key benefits of the app.
• Login screen, OTP input and registered user account verification screens
The target audience or user is a mobile user and a patient or a caregiver. The app screens in the scope of this design challenge aim to provide the user with an easy to understand UI and UX flow for the process of login and user verification and 3 guided introduction screens that inform the user about the benefits of PharmaNeeds app.
