Hoblist

Search Page Redesign

ABOUT THE PROJECT

The project was to redesign this Hoblist page for beautification, an appealing UI/ UX design and ease of navigation for the users. The page features a huge collection of list of top movies related to various categories like movies starring specific actors, belonging to a particular genre, being critically acclaimed, etc.

SCOPE OF WORK

The main purpose of the redesign was to improve the user experience. The original design had an almost impossible to navigate user flow. The user would need to scroll through endless pages in case they were looking for something specific, hence my design incorporates an easy search box, with multiple filters, each with their own drop down menus which displays keywords present in the 1600+ lists, classified by specific categories relating to the lists. While the original design had a dated look with a complicated navigation, the finished redesign is clean & intuitive and has a fresh look to it.
