Visual Con 2021

Home Page Redesign

ABOUT THE PROJECT

The project was an individual design assignment done for Hubilo’s virtual event platform which includes live interactive sessions, breakout rooms, networking lounge, engagement, and gamification features. The project was to design the homepage for the virtual design conference “Visual Con 2021” to optimize for maximum attendee engagement and brand awareness of Hubilo.

SCOPE OF WORK

The home page is the first page that greets the attendee when they enter the event after completing the onboarding flow and needs to be impactful. The page currently displays quick links, important info that an attendee should know about the event and showcases featured sessions and other event constituents like exhibitors and sponsors. The two most important things from an attendee’s perspective in an event are sessions and networking with like-minded folks, everything else is there just to keep them engaged in the event.

The main objective of the design was to ease out the navigation so as to focus on these two important factors for an attendee. The old design was too cluttered and attendee had to look closely for the desired page to navigate to. The finished design is modern and clean with a smooth visual flow that is easy to navigate.
